  • Visual Studio Code 发布王炸更新：Agent 模式上线，支持 MCP 协议！

    本次更新的最大亮点是 Agent 模式 正式在 VSCode Stable 版本中上线。开发者可以通过设置 chat.agent.enabled 启用该功能，并在 Chat 视图的模式选择器中找到它。
